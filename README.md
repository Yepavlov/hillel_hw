Ціль Зібрати форму реєстрації, зробити live- та submit-валідацію, серіалізувати дані.

Форма (поля)

Email
Пароль
Підтвердження пароля
Вік (number)
Місто (select)
Погоджуюсь з умовами (checkbox)
Кнопка “Зареєструватись” (submit)
Кнопка “Очистити” (button)
Валідація

Email: обов’язкове, формат email
Пароль: мін 8 символів, мін 1 цифра, мін 1 літера
Підтвердження: має збігатися з паролем
Вік: обов’язковий, 16–120
Місто: обов’язкове
Checkbox: має бути true
Поведінка

Live-валідація: на input або focusout показувати помилку під полем (або підсвічування + текст).
Submit: якщо є помилки — не відправляти, фокус на перше невалідне поле.
Серіалізація: на submit зібрати дані в об’єкт і показати під формою як JSON.
Clear: кнопка “Очистити” очищає форму + localStorage + UI помилок + JSON результат.
Acceptance criteria

Неможливо відправити форму з невалідними даними.
Після перезавантаження сторінки поля відновлюються.
Під формою відображається JSON з актуальними даними (після submit).
“Очистити” повертає все в початковий стан без артефактів.
Бонус

Індикатор “сила пароля” (weak/medium/strong) за Вашими правилами.
“Показати/сховати пароль”.
